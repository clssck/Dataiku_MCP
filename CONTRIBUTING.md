# Contributing

Thanks for your interest in contributing to `dataiku-mcp`.

## Development Setup

1. Install Node.js 20+.
2. Install dependencies:

```bash
npm install
```

3. Run local checks:

```bash
npm run check
npm test
```

4. Build the package:

```bash
npm run build
```

## Pull Request Process

1. Create a branch from `main`.
2. Keep changes focused and scoped.
3. Add or update tests for behavior changes.
4. Ensure `npm run check` and `npm test` pass.
5. Open a pull request using the PR template.

## Commit Style

- Use clear, concise commit messages.
- Prefer one logical change per commit when practical.

## Reporting Bugs

Please use the bug report issue template and include:

- Steps to reproduce
- Expected behavior
- Actual behavior
- Environment details (Node version, OS, Dataiku version)
